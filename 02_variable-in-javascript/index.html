Variables in JavaScript

A JavaScript variable is just a name of storage location.

And we can give that name in three ways.

  • Using let keyword.
  • Using const keyword.
  • Using var keyword.

Rule for variable Naming

A variable name can contain any of the following characters but must not start with a digit

  • Capital Letter:A to Z
  • small Letter: a to z
  • 0 to 9 (but we can't start variable name with digit)
  • Only two special character that are: Dollar"$" and Underscore "_"
