Data Type in Javascript

There are two type of data type in Javascript

  1. Primitive(value type)
  2. Non-Primitive(reference type)

In this Vedio we'll learn about the Primitive Data type

We will classify Primitive Data tyep into 7 Categories

  1. number
  2. string
  3. boolean
  4. undefined
  5. null
  6. bigint
  7. symbol

There is only one type of non primitive data type that is called object.

object are further divided into three categories.

  • object Literals
  • Arrays
  • Function
